# Bajo Control — Sitio Web

Sitio web institucional para **Bajo Control**, empresa de servicios técnicos
de seguridad, automatización y mantenimiento en 9 de Julio, Buenos Aires.

## 📋 Descripción
MVP desarrollado como trabajo práctico para la materia Diseño y Desarrollo Web.
El sitio presenta los servicios, el perfil del fundador y un formulario de contacto.

## 🛠️ Tecnologías
- HTML5 semántico
- CSS3 (custom + variables)
- Bootstrap 5.3.8
- JavaScript (validación de formulario)

## 📂 Estructura del proyecto
- `index.html` — Página de inicio
- `secciones/` — Servicios, Sobre mí, Contacto
- `assets/` — CSS, JS, imágenes y vendor (Bootstrap local)
- `documentacion/` — Brief UX y documentación del proyecto

## ✅ Criterios de calidad
- Validación HTML/CSS con W3C
- Accesibilidad y performance auditadas con Lighthouse
- Diseño responsive (mobile-first)
